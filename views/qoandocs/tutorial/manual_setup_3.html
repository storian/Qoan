The Run Report. First let's get acquainted with Qoan's debug report. In the application script, add the following code:

use lib qw| /path/to/qoan |; # Replace the path with wherever you installed Qoan. use Qoan::Controller; my $q = Qoan::Controller->new; $q->env( 'allow_http_debug' => 1 ); $q->env( 'allow_http_debug_public' => 1 ); $q->env( 'allow_default_get_action' => 1 ); $q->process_request;
