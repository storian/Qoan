Step 1: Quick Set-up and Test

0. Get Code.
Download the Qoan code from Github: [download page]

1. Create a Qoan applications directory.
Create a directory outside of your web server's public directory, called qoan_apps. This is your {{quickdef}}Qoan applications directory{{/quickdef qoan_applications_dir}}. Unpack the download inside that directory. When you are done there should be a directory inside qoan_apps called Qoan. This is your {{quickdef}}Qoan installation directory{{/quickdef qoan_installation}}.

2. Create a redirection directory.
Create a {{quickdef}} redirection directory{{/quickdef redirection_dir}} in your web server's document root, called qoan. This is just for testing, so you can delete or change it later.

3. Copy redirection files to the new web directory.
These are:

  • Qoan/utility/setup/verify/.htaccess
  • Qoan/utility/setup/verify/qoan_redirect.cfg
  • Qoan/utility/setup/qoan_redirect.cgi

4. Make an edit to the redirector configuration file.
Open qoan_redirect.cfg and replace "/absolute/path/to/qoan_apps" in the "qoan_home" line, with the absolute filesystem path to your {{quickdef}}Qoan applications directory{{/quickdef qoan_applications_dir}}.

5. Test Qoan.
Now visit the URL: http://[your domain]/qoan. If you see the index page for the Qoan documentation, then Qoan is working! Hurray! (If it did not work, hie thee to page 4 of this tutorial.)

If you want to read about what is in the Qoan installation, go here: Qoan installation components.

The next step in this tutorial gives detail about what you've just done. After that there will be different setup configuration options.
